# Collaboration Notes

This repository can be used to practice collaborative GitHub workflows in a controlled environment.

## Co-authored commits

Use co-authored commits only when another real contributor helped with the change. The commit message must include the standard GitHub trailer:

```text
Co-authored-by: Name <github-email@example.com>
```

## Practical checklist

- Confirm the contributor name and GitHub email before committing.
- Keep the pull request small and easy to review.
- Describe the shared contribution in the pull request body.
- Do not add personal data beyond what the contributor explicitly provided for the commit trailer.
